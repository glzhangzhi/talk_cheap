# 创建向量

## 创建行向量

`r = [1 2 3 4 5]`

## 创建列向量

`c = [1;2;3;4;5]`

# 引用向量中的元素

## 单个引用

`r(1)`

`c(2)`

## 引用所有元素

`a(:)`

## 引用一段元素

`b(2:4)`

# 向量运算

```
a = [1 2 3 4 5]
b = [2 3 4 5 6]

c = a + b
d = a - b
e = a * 5
f = a'
```

# 向量堆叠

使用`horzcat`和`vertcat`来堆叠向量，在进行堆叠时要注意向量的维度关系

```
a = [1 2]
b = [3 4]

c = hortcat(a, b)
c = [a,b]

d = vertcat(a, b)
d = [a;b]
```

# 创建具有均匀间隔元素的向量

`v = [a:b:c]`

这样会创建从a开始，到c结束，间隔是b的向量
